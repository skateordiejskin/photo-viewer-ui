# Photo Viewer API

Requirements:
- Node.js >=10

## INSTALLATION STEPS

1) If not installed, install homebrew and yarn

    ```
     /bin/bash -c "(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"

     brew install yarn

    ```
2) Clone repo and cd into folder

3) Install packages

     ` yarn install`

4) start the server

    `yarn serve`
